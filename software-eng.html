Chapter 7: Programming Languages

Word Definition
Machine Language The elemental language of computers. Its composed of digital bianry numbers that look like a long sequence of zeros and ones.Its read by a CPU
Debugging The process of finding and fixing errors or bugs in a source code.
Assembler A program that takes basic computer instructions and converts them into a pattern of bits that the CPU can use to perform basic functions.
Asembly Language 1A type of low-level programming language that is intended to communicate directly with a computers hardware.
Translator A piece of software that handles the translation of source code into a form that can be executed by a processor. (examples: Interpreters and compilers)
Compiler A program that translates a programming language source code into machine code or another programming language.
Interpreter A computer program that is used to directly execute program instructions written using a high-level programming language.
Cross-Platform Software A type of application/program/software that works on various operating systems or devices (platforms).
Natural Language AKA Ordinary Language. A human language as opposed to a machine language or artificial language.
Formal Language An abstract language used ot express definitions, instructions, and logic. It consists of a certain set of character or symbol chains (words), which are formed from other characters.
Machine Independence A term for software that is not dependent on the properties of specific properties of a certain machine. It means it can be used on any machine.
High-level programming language A programming language such as C or Pascal. They enable programmers to write programs that are mostly independent of a particular type of computer. They are considered high-level because they are closer to human languages.
Object-Oriented programming (OOP) A style of programming characterized by the identification of classes of objects closely linked with the methods (functions) with they are associated. Includes Python, Java, C++, and Ruby.
Scripting Language A programming language the employs a high-level construct to interpret and execute one command at a tiem. Usually easier to learn and faster to code in languages like C+ and C++. Its a language that is interpreted and translated into machine code when the code is run rather than beforehand.
Data Types:
  • Integer- A whole number
  • Float- A function or reusable code in the Python programming language that converts values into floating point numbers. AKA fractional numbers, or numbers with decimals.
  • String- A sequence of characters used to represent text.
Primitive Data type a data type that is built into a programming language, or one that could be characterized as a basic structure for building more sophisticated data types.
Data Structure The format in which data is stored, organized, processed, and retrieved. the four basic types are linear, tree, hash, and graph.
Python Collection Data Structures
  • List- Lists are used to store multiple items in a single variable. They are ordered, changeable, and allow duplicate values.
  • Tuple- Can also store multiple items in a single variable. It is ordered and unchangeable.
  • Set- Can also store multiple items in a single variable. It is unordered, unchangeable (can add and remove items), and unindexed.
  • Dictionary- Used to store data values in key value pairs. A collections that is ordered, changeable, and doesnt allow duplicates.
Function A chunk of code that can be used over and over rather than being rewritten.
Local Variable A variable that is only accessible within a specific part of a program.
Global Variable A variable that is defined outside of a function, and it is accessible to all areas of a program.
Source Program AKA source file, or source code. A text file that contains instructions written in a high-level language that cant be executed by a processos without additional steps.
Object Program An abstract data type created by a developer.
