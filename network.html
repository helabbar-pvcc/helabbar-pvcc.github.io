Operating Systems Terminology

Term Definition
Network Classifications LAN MAN WAN
Job Que An ordered list of tasks to be carried out by an operating system.
FIFO "First In, First Out." A type of data handeling. Its like the processes are standing in line.
Interactive Processing A feature that allowed a program being executed to carry on a dialogue with the user through remote terminals. AKA A computer program directs the system to alternately accept input and respond to the input.
Time-sharing The operation of a computer system by several users for different operations at the same time.
Multiprogramming Running multiple tasks on one computer system.
Application Software Application software is task-specific computer programs that can assist personal, educational or business tasks. They connect a computer user to the system software
System Software The computer program that runs a computers hardware and application programs.
Utility Software All systems and programs on a computer system that maintain its functionality.
Graphical User Interface (GUI) Files/Programs represented by icons and menus.
Shell The interface to the operating system, which is the outermost layer of the operating system. It is interacted with using keyboard commands rather than GUIs.
Kernel The internal part of an operating system that contains the software components that perform basic functions required by the computer installation
Device Driver A set of files that tells a piece of hardware how to function by communicating with a computer's operating system
File Manager A software program that helps users manage their files on their computers.
Memory Manager The software that coordinates the machines use of main memory.
Virtual Memory A method that computers use to manage storage space to keep the system running quickly and efficiently. Ex: Transfering data from RAM to a hard drive.
Boot Strapping The process that transfers the operating system from mass storage into main memory
ROM "Read-Only Memory." A type of computer storage that is kept permanently as initially set. It cant be altered.
Deadlock A condition where two or more processes are blocked because each is waiting for the other.
Spooling SPOOL:"Similtaneous Peripheral Operations Online" To read/store a document or task list so it can be printed or processed at a more convenient time.
Super User (or Administrator) An individual that is the resident expert for a system or program.
Multi-factor Authentication An authentication method that requires the user to provide two or more verification factors to gain access to a resource.
