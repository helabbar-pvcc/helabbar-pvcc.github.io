Network Models An ordered list of tasks to be carried out by an operating system.
Protocols "First In, First Out." A type of data handeling. Its like the processes are standing in line.
Combining Networks A feature that allowed a program being executed to carry on a dialogue with the user through remote terminals. AKA A computer program directs the system to alternately accept input and respond to the input.
Client/server model: The operation of a computer system by several users for different operations at the same time.
Internet Addressing Running multiple tasks on one computer system.
IP Adress Application software is task-specific computer programs that can assist personal, educational or business tasks. They connect a computer user to the system software
Internet Protocols The computer program that runs a computers hardware and application programs.
World Wide Web All systems and programs on a computer system that maintain its functionality.
Internet four-layer software Files/Programs represented by icons and menus.
Internet is: The interface to the operating system, which is the outermost layer of the operating system. It is interacted with using keyboard commands rather than GUIs.
Cybersecurity The internal part of an operating system that contains the software components that perform basic functions required by the computer installation
Protection A set of files that tells a piece of hardware how to function by communicating with a computer's operating system
